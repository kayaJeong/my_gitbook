YUM 사용법


1. YUM 이란?

  • Yellowdog Updater Modified 명령어는 CentOS에서 패키지 관리를 하는 명령어
  • /etc/yum.conf에 설정 내용, /etc/yum.repos.d/ 디렉토리에 저장된 저장소 파일에 지정된 서버로부터 업데이트된 패키지들을 검사하고 다운로드하여 설치함
  • 설치, 업데이트, 삭제 시 의존성을 검사하여 관련된 패키지를 설치, 업데이트, 삭제를 함께 진행함
