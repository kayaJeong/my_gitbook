MarkDown

Markdown 에 대해서

마크다운이란

  • 서식이 있는 문서를 작성하는 데 사용되며, 일반 마크업 언어에 비해 문법이 쉽고 간단함.
  • HTML과 리치 텍스트(RTF) 등 서식 문서로 쉽게 변환되기 때문에 응용 소프트웨어와 함께 배포되는 README 파일이나 온라인 게시물 등에 많이 사용

1. 제목


1-1. 큰 제목 : 제목을 적고 바로 아래 줄에 "===" 입력

큰 제목

=============


1-2. 작은 제목 : 제목 적고 바로 아래 줄에 "-----" 입력

작은 제목 

작은 제목 
-------------
